# Spacing

Spacing in Petrichor follows a consistent scale based on multiples of 4px. This creates visual rhythm and makes layout decisions predictable. Components never use arbitrary pixel values — they reference spacing tokens.

---

## The scale

The spacing scale ranges from 1px (borders only) to 96px (major layout divisions). Most day-to-day work uses `sm` (16px) and `md` (24px) — the workhorses of the system.

## When to use what

| Scale | Value | Best for |
| ----- | ----- | -------- |
| `px` | 1px | Borders only — never for spacing |
| `3xs` | 4px | Icon gaps, inline badge spacing |
| `2xs` | 8px | Button/input vertical padding, tight groups |
| `xs` | 12px | Button/input horizontal padding |
| `sm` | 16px | Card padding, form gaps — **default choice** |
| `md` | 24px | Section gaps, card-to-card — **default choice** |
| `lg` | 32px | Large section padding |
| `xl` | 48px | Page section divisions |
| `2xl` | 64px | Hero sections, major breaks |
| `3xl` | 96px | Layout margins |

**When in doubt, reach for `sm` or `md`.** They handle 80% of spacing decisions correctly.

---

## Philosophy

**Rhythm over randomness.** A 4px base unit creates harmonious proportions. Elements align naturally because every dimension is a multiple of the base. Arbitrary values (13px, 17px, 22px) break the rhythm.

**Density is a brand choice.** Raindrop is data-dense — information-rich dashboards benefit from compact spacing. Dusk Rose is editorial — generous whitespace lets prose breathe. The same components, configured with different tokens.

**Gap over margin.** When laying out siblings in flex or grid containers, use `gap` instead of margins. It's more predictable (no collapsing margins), easier to adjust, and the spacing stays with the container rather than scattering across children.

**Components don't set external margin.** A Button doesn't know how far it should be from the next element — that's the parent's job. Components control their internal spacing (padding, gap). Layout spacing (margin between components) belongs to the container.
